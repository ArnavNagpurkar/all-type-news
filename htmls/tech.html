WINDOWS 11 NEW FEATURES AND CHANGES

How to upgrade to Windows 11 21H2 for free via Windows Update

The easiest and safest method to upgrade for free to Windows 11 (version 21H2) from Windows 10 is using Windows Update. The reason is that through Windows Update settings, you will get a notification letting you whether or not version 21H2 is compatible with your device, and you can through the installation with a few clicks knowing that all your files, most settings, and applications will migrate automatically. The experience is similar to installing a cumulative update, but with the difference that this process will take longer to complete (about an hour) because of the download size (around 4 to 5GB) and because the process will perform a complete installation.
